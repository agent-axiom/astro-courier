# Wow Run V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add compact perks, risk gates, mini-goal readouts, and extra arcade feedback without making the UI text-heavy.

**Architecture:** Simulation owns deterministic perk and risk gate state. GameShell passes the selected perk into world creation and publishes compact HUD values. Renderer draws risk gates from snapshots. React adds a small preflight selector and mini-goal chips.

**Tech Stack:** TypeScript, Vitest, Vite React, Pixi.js, pnpm monorepo.
